## Пользовательская Грамматика (Контекстно-Свободная)

| Нетерминал | Правило | Описание |
| :--- | :--- | :--- |
| **P** | `S EOF` | Программа состоит из списка операторов, за которым следует конец файла. |
| **S** | `Stmt S` | Список операторов (один или более). |
| | `ε` | Пустой список операторов. |
| **Stmt** | `INT ID ;` | Объявление переменной. |
| | `ID = E ;` | Присваивание. |
| | `PRINT E ;` | Вывод выражения. |
| | `IF (C) {S} StmtElse` | Условный оператор. |
| | `WHILE (C) {S}` | Цикл. |
| **StmtElse** | `ELSE {S}` | Ветка ELSE. |
| | `ε` | Отсутствие ветки ELSE. |
| **C** | `E RELOP E` | Условие сравнения. |
| **E** | `E (+ | -) T` | Выражение с операциями сложения/вычитания (левая рекурсия). |
| | `T` | Терм. |
| **T** | `T * F` | Терм с операцией умножения (левая рекурсия). |
| | `F` | Фактор. |
| **F** | `(E)` | Выражение в скобках. |
| | `ID` | Идентификатор (переменная). |
| | `INT_LITERAL` | Целочисленный литерал. |

**Терминалы (Токены):**

* `INT`, `ID`, `PRINT`, `IF`, `ELSE`, `WHILE`
* `=`, `;`, `(`, `)`, `{`, `}`
* `+`, `-`, `*`
* **RELOP**: `==`, `!=`, `<`, `>`
* `INT_LITERAL`, `EOF`
